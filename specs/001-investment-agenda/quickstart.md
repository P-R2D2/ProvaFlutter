# Quickstart: Investment Agenda

## Prerequisites
- Flutter SDK (>= 3.0.0)
- Dart SDK (>= 3.0.0)

## Setup
1. Clone the repository.
2. Run `flutter pub get` to install dependencies:
   - `provider`
   - `go_router`
   - `uuid`
   - `intl`

## Running the App
```bash
flutter run
```

## Features for Testing
- **Login**: Use `admin` / `admin` (hardcoded).
- **Add**: Tap the FAB on the dashboard.
- **Delete**: Swipe or tap the delete icon on a card and confirm the dialog.
